 # PocketChef
 PocketChef is an intelligent cooking companion app built with modern Android technologies. It helps users manage their ingredients, find recipes based on what they have using AI, and get daily cooking tips.


 ## ‚ú® Features
 *   **AI-Powered Recipe Generation**: Suggests recipes based on the ingredients you currently have in your inventory using Gemini AI.
 *   **Smart Ingredient Management**: Keep track of your pantry. Add or remove ingredients easily.
 *   **Ingredient Suggestions**: Smart auto-completion/suggestions when adding new ingredients.
 *   **Daily Inspiration**: Receive daily recipe reminders and cooking tips.
 *   **Favorites**: Save your favorite AI-generated recipes for later.
 *   **Cloud Sync**: Sync your ingredients and preferences across devices using Firebase.
 *   **Beautiful UI**: Clean and modern interface built with Jetpack Compose, featuring both Light and Dark themes.
